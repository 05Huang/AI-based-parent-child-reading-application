# 图片资源目录

```
static/images/
├── icons/           # SVG 图标
│   ├── common/      # 通用图标
│   ├── parent/      # 家长端专用图标
│   └── child/       # 孩子端专用图标
├── avatars/         # 头像图片 (PNG)
├── books/           # 书籍相关图片 (PNG)
│   ├── covers/      # 书籍封面
│   └── content/     # 书籍内容图片
└── backgrounds/     # 背景图片 (PNG/JPG)
```

## 图标命名规范
- 使用小写字母
- 单词间用连字符（-）分隔
- 表明用途，例如：`icon-parent-user.svg`

## 图片格式说明
- 图标类素材优先使用 SVG 格式
- 照片、复杂图片使用 PNG 格式
- 大型背景图可使用适当压缩的 JPG 格式

## 图标尺寸建议
- SVG 图标：保持原始尺寸，通过 CSS 控制大小
- 头像图片：建议 200x200 像素
- 书籍封面：建议 400x600 像素
- 其他图片：根据实际需求确定尺寸
