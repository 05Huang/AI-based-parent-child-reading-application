                        测试数据生成工具 - 文件说明

📁 核心文件
-----------

1. generate_test_content.py [核心脚本]
   功能：
   - 从Unsplash、Pexels爬取图片和视频
   - 上传到MinIO对象存储
   - 生成测试文章和视频数据
   - 插入MySQL数据库
   
   特点：
   - 支持交互式配置生成数量
   - 自动生成推荐系统所需字段
   - 详细的日志输出便于调试
   - 支持配置文件和环境变量

2. check_environment.py [环境检查脚本]
   功能：
   - 检查Python版本和依赖包
   - 检查MySQL连接和数据库表
   - 检查MinIO连接和bucket
   - 检查API配置
   
   使用：运行 python check_environment.py


📁 配置文件
-----------

3. test_data_requirements.txt [依赖包列表]
   内容：
   - requests==2.32.5
   - PyMySQL==1.1.0
   - minio==7.2.0
   
   使用：pip install -r test_data_requirements.txt

4. config.example.json [配置示例]
   说明：
   - 复制为config.json使用
   - 包含MinIO、MySQL、API密钥配置
   - 支持自定义所有连接参数


📁 启动脚本
-----------

5. run_generate_test_data.bat [Windows启动脚本]
   功能：
   - 自动检查Python环境
   - 自动安装依赖包（使用清华镜像加速）
   - 自动执行环境检查
   - 启动数据生成脚本
   
   使用：双击运行

🎯 推荐使用流程
---------------

首次使用：
  1. 阅读"快速开始.txt"
  2. 双击运行"run_generate_test_data.bat"
  3. 根据提示输入少量数据进行测试（5篇文章、2个视频）
  4. 观察执行日志，确认环境配置正确

配置优化（可选）：
  5. 参考"config.example.json"创建"config.json"
  6. 配置Unsplash和Pexels的API密钥
  7. 重新运行，生成更多测试数据

批量生成：
  8. 确认环境正常后，可以生成大量数据
  9. 建议：50-100篇文章，10-20个视频

📋 文件依赖关系
---------------

run_generate_test_data.bat
  └─→ check_environment.py [检查环境]
        ├─→ 检查依赖包
        ├─→ 检查MySQL
        └─→ 检查MinIO

        ├─→ config.json [可选配置]

        └─→ MySQL [插入数据]

🔧 自定义修改
-------------

修改文章模板：
  编辑 generate_test_content.py
  找到 generate_article_content() 方法
  修改 paragraphs 列表

修改标题生成规则：
  编辑 generate_test_content.py
  找到 generate_title() 方法
  修改 title_templates 和 topics

修改标签池：
  编辑 generate_test_content.py
  找到 generate_tags() 方法
  修改 tag_pool

修改爬取关键词：
  编辑 generate_test_content.py
  找到 generate_articles() 和 generate_videos() 方法
  修改 keywords 列表

⚙️ 技术细节
-----------

MinIO路径规范：
  图文封面: /qz-sns/image/article/{id}/article_cover.jpg
  文章图片: /qz-sns/image/article/{id}/article_image_{n}.jpg
  视频封面: /qz-sns/video/resource/{id}/video_cover.jpg
  视频文件: /qz-sns/video/resource/{id}/video.mp4

数据库表：
  - content: 主内容表（图文和视频）
  - content_image: 文章图片关联表
  - content_category: 内容分类表

推荐系统字段：
  - recommendation_score: 综合推荐分数
  - popularity_score: 热度得分
  - click_through_rate: 点击率
  - completion_rate: 完成率
  - content_length: 内容长度
  - last_recommended: 最后推荐时间
