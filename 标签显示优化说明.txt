===============================================================================
                        首页标签显示优化说明
===============================================================================

需求描述
--------
首页精选推荐和热门文章的标签栏由于大小限制，不能展示所有标签。
需要只显示前两个标签，避免溢出和布局混乱。


优化方案
--------

创建标签格式化函数，智能处理标签显示：
1. 只显示前两个标签
2. 使用 · 分隔符连接
3. 处理空标签和空格
4. 提供默认值


技术实现
--------

1. 创建格式化函数
   ```javascript
   const formatTags = (tags) => {
     if (!tags) return '推荐阅读'
     
     // 按逗号分隔标签
     const tagArray = typeof tags === 'string' ? tags.split(',') : tags
     
     // 过滤空标签并去除空格
     const cleanTags = tagArray
       .map(tag => tag.trim())
       .filter(tag => tag.length > 0)
     
     if (cleanTags.length === 0) return '推荐阅读'
     
     // 只取前两个标签
     const displayTags = cleanTags.slice(0, 2)
     
     // 用 · 分隔
     return displayTags.join(' · ')
   }
   ```

2. 在模板中使用
   ```vue
   <!-- 精选推荐 -->
   <text class="book-age">{{ formatTags(item.tags) }}</text>
   
   <!-- 热门文章 -->
   <text class="hot-book-tag">{{ formatTags(item.tags) }}</text>
   ```


显示效果
--------

原始标签数据格式（数据库存储）：
  "亲子阅读,家庭教育,儿童成长,学习方法,阅读习惯"

优化前显示：
  ❌ "亲子阅读,家庭教育,儿童成长,学习方法,阅读习惯"
  ❌ 文字过长，超出容器
  ❌ 显示混乱，影响美观

优化后显示：
  ✅ "亲子阅读 · 家庭教育"
  ✅ 只显示前两个，简洁美观
  ✅ 使用优雅的分隔符
  ✅ 自适应容器大小

自定义配置
----------

如果需要调整显示数量：

显示3个标签：
```javascript
const displayTags = cleanTags.slice(0, 3)
```

显示1个标签：
```javascript
const displayTags = cleanTags.slice(0, 1)
```

自定义分隔符：
```javascript
// 使用空格
return displayTags.join(' ')

// 使用逗号
return displayTags.join(', ')

// 使用 |
return displayTags.join(' | ')

// 使用 /
return displayTags.join(' / ')
```

添加省略号提示：
```javascript
const displayTags = cleanTags.slice(0, 2)
const hasMore = cleanTags.length > 2

return displayTags.join(' · ') + (hasMore ? '...' : '')
```


样式说明
--------

标签容器样式保持不变：
```css
.book-age {
  font-size: 22rpx;
  color: #3b82f6;
  background: linear-gradient(135deg, #dbeafe 0%, #bfdbfe 100%);
  padding: 6rpx 16rpx;
  border-radius: 12rpx;
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}
```

即使只显示2个标签，仍然保留：
- overflow: hidden（防止意外溢出）
- text-overflow: ellipsis（超长显示省略号）
- white-space: nowrap（单行显示）

性能说明
--------

formatTags 函数：
- 时间复杂度：O(n)，n为标签数量
- 空间复杂度：O(n)
- 性能影响：极小，可忽略

每次渲染都会调用，但：
- 标签数量通常 ≤ 10
- 字符串操作简单
- 无异步操作
- 无DOM操作
